E-commerce Website

This is a simple e-commerce website built with React. Users can browse products and manage their cart.

# Tech Stack Used

React,
Vite,
Bootstrap.

# Features

Product Listing,
Cart Functionality,
Search Bar,
Login System.

# How to Run the Project
npm install,
npm run dev.
